Resource architecture

Lecture materials can be downloaded here.

There is a separate resource page for every course.

Lecture delivery

PowerPoint and PDF slide decks are published by week, module or topic and linked directly from the relevant course page.

Code and reproducibility

R scripts, R Markdown files, Python scripts and Jupyter notebooks are stored separately, described clearly and linked to the lecture or practical they support.

Data and applied practice

CSV, Excel and SPSS datasets, worked examples, practical exercises and non-confidential assessment support are published in a structured learning library.

This site is a ready live course-resource platform.

The official outlines are downloadable from their course pages. Slides and scripts are added to the relevant course rather than placed in one unstructured download list.
