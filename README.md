# Collins Baffour Kyei — Academic Portfolio & Teaching Resource Hub

This repository contains the source files for the personal academic website of **Collins Baffour Kyei**, an economist, doctoral researcher, lecturer, researcher and educator at the **University of Cape Coast, Ghana**.

The website brings together my academic profile, research agenda, peer-reviewed publications, teaching portfolio, research software, educational writing, academic leadership, professional training and capacity-building activities. It also serves as a growing digital resource hub for selected undergraduate and postgraduate courses that I teach or support.

My work sits at the intersection of **macroeconomics, monetary economics, financial economics, data science, machine learning and evidence-informed public engagement**. I treat econometrics as an essential part of applied economic research and use quantitative methods as tools for answering policy-relevant questions rather than as a separate professional identity.

> **Economist, Researcher & Educator**

My research and teaching activities span several interconnected areas:

## Course Resource Architecture

Teaching resources are organised under:

```text
assets/
└── courses/
    ├── eco107/
    ├── ecf206/
    ├── est206/
    ├── eco305/
    ├── eco306/
    ├── eco324/
    ├── dis814/
    └── qrm/
```

Each course directory may contain:

```text
slides/        Lecture slides and presentation materials
r/             R scripts and R Markdown resources
python/        Python scripts and Jupyter notebooks
data/          Teaching and demonstration datasets
practicals/    Exercises, laboratory tasks and worked examples
assessments/   Published learning and revision resources
```

Course metadata is maintained in:

```text
assets/data/course-manifest.json
```

This architecture allows the teaching-resource library to expand while maintaining consistent organisation across courses.
